## Usage

### Run Example

```sh
$ git clone https://github.com/ianchen0119/GO-CPSV.git
$ cd GO-CPSV
$ go get github.com/ianchen0119/GO-CPSV/cpsv
$ go run main.go
```

### Go module

```sh
$ cd YOUR_PROJECT
$ go get github.com/ianchen0119/GO-CPSV/cpsv
```

in your program:
```go=
import (
  "fmt"
  "github.com/ianchen0119/GO-CPSV/cpsv"
  // ...
)

// ...
```
